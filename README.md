# 15puzzle

ブラウザで遊べるスライドパズルです。PC とタブレットの両方で操作しやすいように、レスポンシブ UI で実装しています。

## 遊び方

1. `盤面サイズ` で 3×3（8パズル）/ 4×4（15パズル）/ 5×5（24パズル）を選択
2. `シャッフル` を押してゲーム開始
3. 空きマスに隣接するタイルをタップ / クリックして移動
4. 数字を順番どおりに並べるとクリア

## ローカルで実行

静的ファイルだけで動くため、以下で起動できます。

```bash
python3 -m http.server 8000
```

`http://localhost:8000` を開いてプレイしてください。
